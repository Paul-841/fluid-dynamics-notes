# Section 7.2: Buckingham Pi Theorem

> **Core theme:** The Buckingham pi theorem is the **foundational theorem of dimensional analysis**. It answers two critical questions: (1) Can we always reduce the number of variables? (2) How many dimensionless products result?

***

## 1. The Theorem Statement

> **If an equation involving $k$ variables is dimensionally homogeneous, it can be reduced to a relationship among $k - r$ independent dimensionless products, where $r$ is the minimum number of reference dimensions required to describe the variables.**

### Key Points:

- **$k$** = total number of physical variables in the problem
- **$r$** = minimum number of reference (base) dimensions needed to express all $k$ variables
  - Typically $M$ (mass), $L$ (length), $T$ (time) → **$r = 3$** in MLT system
  - Or $F$ (force), $L$, $T$ → also **$r = 3$** in FLT system
- **$k - r$** = number of independent dimensionless $\Pi$ groups that result
- The $\Pi$ groups are **dimensionless products** — each has the form $\Pi = V_1^{a_1} V_2^{a_2} \cdots V_m^{a_m}$

***

## 2. Example: Pipe Flow (Revisited)

### Step 1: List the variables

| Variable | Symbol | Dimensions (FLT) |
|----------|--------|-------------------|
| Pressure drop per length | $\Delta p_\ell$ | $FL^{-3}$ |
| Pipe diameter | $D$ | $L$ |
| Fluid density | $\rho$ | $FL^{-4}T^2$ |
| Fluid viscosity | $\mu$ | $FL^{-2}T$ |
| Mean velocity | $V$ | $LT^{-1}$ |

$k = 5$ variables

### Step 2: Determine $r$

The variables involve: $F$, $L$, $T$ — three base dimensions → $r = 3$

### Step 3: Number of $\Pi$ groups

$$k - r = 5 - 3 = 2$$

Therefore the relationship $\Delta p_\ell = f(D, \rho, \mu, V)$ can be reduced to:

$$\Pi_1 = \phi(\Pi_2)$$

where:
- $\Pi_1 = \dfrac{D \Delta p_\ell}{\rho V^2}$ — dimensionless pressure drop (Euler number form)
- $\Pi_2 = \dfrac{\rho V D}{\mu}$ — Reynolds number

---

## 3. Choosing Reference Dimensions

Two commonly used systems are equivalent via Newton's second law:

| System | Base Dimensions | Common Usage |
|--------|----------------|--------------|
| **MLT** | Mass $M$, Length $L$, Time $T$ | Physics, general science |
| **FLT** | Force $F$, Length $L$, Time $T$ | Engineering (especially fluids/structures) |

**Conversion:** $F \doteq MLT^{-2}$ (Newton's second law)

> Use whichever is more convenient. The $\Pi$ groups come out the same either way.

---

## 4. Important Limitations

| Limitation | Explanation |
|------------|-------------|
| **Variables must be complete** | If you leave out an important variable, the $\Pi$ groups will be wrong |
| **The equation must be dimensionally homogeneous** | Every term must have the same dimensions — otherwise dimensional analysis cannot apply |
| **Doesn't tell you the functional form** | $\phi$ must still be determined by experiment or computation |
| **Choice of repeating variables is not unique** | Different choices give different-looking $\Pi$ groups, but they're mathematically equivalent |

---

## Key Takeaways

| Concept | Why It Matters |
|---------|---------------|
| $k - r$ = number of $\Pi$ groups | Predicts the simplification factor before any work is done |
| $r$ is the **minimum** reference dimensions | Some problems involve fewer dims (e.g., $r=2$ for certain flows) |
| Dimensional homogeneity is **required** | Check your equations! |
| The method is systematic | Easy to apply once variables are identified |
