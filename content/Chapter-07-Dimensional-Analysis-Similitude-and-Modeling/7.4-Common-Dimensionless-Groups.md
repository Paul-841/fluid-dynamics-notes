# Section 7.4: Common Dimensionless Groups in Fluid Mechanics

> **Core theme:** Certain dimensionless groups appear repeatedly across fluid mechanics. Each measures the **ratio of two physical effects** — knowing which group is relevant tells you which physics dominates.

***

## 1. The Most Important Dimensionless Groups

### Reynolds Number — $\text{Re} = \dfrac{\rho V L}{\mu} = \dfrac{V L}{\nu}$

| Aspect | Detail |
|--------|--------|
| **Ratio** | Inertia forces / Viscous forces |
| **Significance** | Laminar vs turbulent transition; viscous vs inviscid regimes |
| **When it matters** | Nearly always — pipe flow, boundary layers, drag, mixing |
| **Typical range** | $\text{Re} < 2300$ (pipe laminar), $\text{Re} > 4000$ (turbulent) |

### Froude Number — $\text{Fr} = \dfrac{V}{\sqrt{g L}}$

| Aspect | Detail |
|--------|--------|
| **Ratio** | Inertia forces / Gravity forces |
| **Significance** | Free-surface flows, waves, open channels, ships |
| **When it matters** | When a free surface or density interface exists |
| **Typical range** | $\text{Fr} < 1$ (subcritical), $\text{Fr} > 1$ (supercritical) |

### Euler Number (Pressure Coefficient) — $\text{Eu} = \dfrac{\Delta p}{\rho V^2}$

| Aspect | Detail |
|--------|--------|
| **Ratio** | Pressure forces / Inertia forces |
| **Significance** | Pressure drop, cavitation, flow around bodies |
| **When it matters** | Whenever pressure difference is the quantity of interest |
| **Typical values** | $C_p = 1$ at stagnation point (inviscid) |

### Mach Number — $\text{Ma} = \dfrac{V}{c}$

| Aspect | Detail |
|--------|--------|
| **Ratio** | Flow speed / Speed of sound |
| **Significance** | Compressibility effects |
| **When it matters** | $\text{Ma} > 0.3$ — density changes become non-negligible |
| **Regimes** | Subsonic ($<0.8$), Transonic ($0.8$–$1.2$), Supersonic ($1.2$–$5$), Hypersonic ($>5$) |

### Weber Number — $\text{We} = \dfrac{\rho V^2 L}{\sigma}$

| Aspect | Detail |
|--------|--------|
| **Ratio** | Inertia forces / Surface tension forces |
| **Significance** | Droplets, bubbles, jets, thin films |
| **When it matters** | Small scales, high curvature interfaces, atomization |
| **Typical range** | $\text{We} \ll 1$ (surface tension dominates), $\text{We} \gg 1$ (inertia dominates) |

---

## 2. Quick Reference: Which Number for Which Problem?

| Problem Type | Primary $\Pi$ Group | Secondary Group(s) |
|-------------|-------------------|--------------------|
| Pipe/duct flow (incompressible) | $\text{Re}$ | $\text{Eu}$ (pressure drop) |
| Pipe/duct flow (compressible) | $\text{Ma}$, $\text{Re}$ | $\text{Eu}$ |
| Open channel / spillway | $\text{Fr}$ | $\text{Re}$ (if viscous effects matter) |
| Ship / surface vessel resistance | $\text{Fr}$, $\text{Re}$ | $\text{We}$ (spray) |
| Airfoil / wing | $\text{Ma}$, $\text{Re}$ | $\text{Eu}$ ($C_p$) |
| Pump / turbine | $\text{Re}$, specific speed | $\text{Eu}$ (head coefficient) |
| Droplet / bubble dynamics | $\text{We}$, $\text{Re}$ | $\text{Oh} = \frac{\sqrt{\text{We}}}{\text{Re}}$ |

---

## 3. Dimensionless Groups for Different Flow Regimes

### Viscous-dominated (low Re):
- $\text{Re}$ is the **only** important parameter
- Pressure drop scales linearly with $V$ (Stokes flow)

### Gravity-dominated (free surface):
- $\text{Fr}$ is primary; $\text{Re}$ matters for boundary layers
- Wave speed: $c = \sqrt{g h}$ (shallow water)

### Compressibility-dominated (high Ma):
- $\text{Ma}$ is primary
- Shock waves, choking, expansion fans

### Surface-tension-dominated (small scale):
- $\text{We}$ or $\text{Oh}$ (Ohnesorge number)
- Capillary waves, jet breakup

---

## Key Takeaways

| Group | $\Pi$ | Used For |
|-------|-------|----------|
| $\text{Re}$ | $\rho V L / \mu$ | Viscous vs inertia |
| $\text{Fr}$ | $V / \sqrt{gL}$ | Gravity vs inertia |
| $\text{Eu}$ | $\Delta p / \rho V^2$ | Pressure vs inertia |
| $\text{Ma}$ | $V / c$ | Compressibility |
| $\text{We}$ | $\rho V^2 L / \sigma$ | Surface tension vs inertia |
