# Section 8.4: Dimensional Analysis of Pipe Flow

> **Core theme:** The **Moody chart** is the culmination of dimensional analysis applied to pipe flow. It condenses thousands of experiments into a single graph: friction factor $f$ as a function of Reynolds number $\text{Re}$ and relative roughness $\varepsilon/D$. This is one of the most-used figures in all of engineering.

***

## 1. Dimensional Analysis Setup

From the variables influencing pressure drop in a pipe:

$$\Delta p = f\!(D,\ \ell,\ \varepsilon,\ V,\ \rho,\ \mu)$$

Buckingham Pi produces:

$$\frac{\Delta p}{\frac{1}{2}\rho V^2} \cdot \frac{D}{\ell} = \phi\!\left(\text{Re},\ \frac{\varepsilon}{D}\right)$$

Define the **Darcy friction factor** $f$:

$$f = \frac{\Delta p}{\frac{1}{2}\rho V^2} \cdot \frac{D}{\ell} = \phi\!\left(\text{Re},\ \frac{\varepsilon}{D}\right)$$

> 📌 This is the pipe flow application of the Buckingham Pi theorem from [[../Chapter-07-Dimensional-Analysis-Similitude-and-Modeling/7.2-Buckingham-Pi-Theorem\|Ch07 §7.2]] — $k=7$ variables → $k-r=4$ $\Pi$ groups, but $D/\ell$ is absorbed into $f$ and gravity is for inclined pipes.

***

## 2. The Moody Chart — Four Regimes

```
log f
  │
  │  Laminar: f = 64/Re  ─┐
  │                        ├── Transition region
  │                         │   ┌── Smooth turbulent
  │                          │  │
  │                          │  │   ╲  ╲ ε/D = 0.05
  │                          │  │    ╲  ╲
  │                          │  │     ╲  ╲ ε/D = 0.001
  │                          │  │      ╲  ╲
  │                          │  │       ╲  ╲ ε/D = 0.00001
  │                          │  │        ╲── Fully rough
  └──────────────────────────┴──┴─────────────────
                         2100  4000            log Re
```

### Regime 1: Laminar ($\text{Re} < 2100$)

$$f = \frac{64}{\text{Re}} \qquad \text{(exact, from §8.2)}$$

- $f$ is **independent of roughness**
- Single straight line on log-log plot (slope = $-1$)

### Regime 2: Transitional ($2100 < \text{Re} < 4000$)

- Unpredictable — avoid design in this range
- $f$ can jump by factor of 2–3

### Regime 3: Turbulent, Smooth Pipe

$$f = f(\text{Re}) \quad\text{only}$$

**Blasius correlation** ($4000 < \text{Re} < 10^5$):

$$f = \frac{0.316}{\text{Re}^{1/4}}$$

**Prandtl smooth-pipe law** (implicit):

$$\frac{1}{\sqrt{f}} = 2.0\log_{10}(\text{Re}\sqrt{f}) - 0.8$$

### Regime 4: Turbulent, Fully Rough

At high $\text{Re}$, $f$ depends **only on $\varepsilon/D$**:

$$\frac{1}{\sqrt{f}} = 2.0\log_{10}\!\left(\frac{3.7}{\varepsilon/D}\right)$$

The curves flatten — increasing $\text{Re}$ no longer changes $f$.

***

## 3. The Colebrook Equation (Unifies All Turbulent Regimes)

Covering smooth, transitional-rough, and fully rough turbulent flow:

$$\frac{1}{\sqrt{f}} = -2.0\log_{10}\!\left(\frac{\varepsilon/D}{3.7} + \frac{2.51}{\text{Re}\sqrt{f}}\right)$$

| Term | Significance |
|------|-------------|
| $\varepsilon/D / 3.7$ | Roughness dominates at high $\text{Re}$ |
| $2.51/(\text{Re}\sqrt{f})$ | Viscous sublayer effect dominates at low $\text{Re}$ |

> ⚠️ The Colebrook equation is **implicit** in $f$ — requires iteration. The explicit **Swamee–Jain** approximation (1976) gives $f$ within 1%:
> $$f = \frac{0.25}{\left[\log_{10}\!\left(\frac{\varepsilon/D}{3.7} + \frac{5.74}{\text{Re}^{0.9}}\right)\right]^2}$$

***

## 4. Roughness Values

| Material | $\varepsilon$ (mm) |
|----------|-------------------|
| Drawn tubing (glass, plastic) | 0.0015 |
| Commercial steel | 0.045 |
| Galvanized iron | 0.15 |
| Cast iron | 0.26 |
| Concrete | 0.3–3.0 |
| Riveted steel | 0.9–9.0 |

***

## 5. Noncircular Ducts — Hydraulic Diameter

For noncircular cross-sections, replace $D$ with **hydraulic diameter** $D_h$:

$$D_h = \frac{4A}{P} = \frac{4 \times \text{cross-sectional area}}{\text{wetted perimeter}}$$

| Shape | $D_h$ |
|-------|------|
| Circle, diameter $D$ | $D$ |
| Square, side $a$ | $a$ |
| Rectangle $a \times b$ (full) | $\frac{2ab}{a+b}$ |
| Annulus, $D_o$ outer, $D_i$ inner | $D_o - D_i$ |

**Laminar $f$: $f = C/\text{Re}_{D_h}$** where $C$ depends on shape (not $64$!). Turbulent Moody chart works reasonably with $D_h$.

***

## Key Takeaways

| Regime | $f$ = | Depends on |
|--------|-------|-----------|
| Laminar | $64/\text{Re}$ | $\text{Re}$ only |
| Smooth turbulent | Blasius / Prandtl | $\text{Re}$ only |
| Transitional rough | Colebrook | $\text{Re}$, $\varepsilon/D$ |
| Fully rough | von Kármán | $\varepsilon/D$ only |
