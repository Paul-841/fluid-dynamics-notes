# Section 7.3: Determination of Pi Terms

> **Core theme:** The Buckingham pi theorem tells us *how many* dimensionless groups exist. This section provides the **step-by-step procedure** for actually finding them.

***

## 1. Step-by-Step Procedure

### Step 1: List All Variables

Write down every physical variable that influences the phenomenon. Include only **independent variables** (don't count the same thing twice).

Example (pipe flow): $\Delta p_\ell,\ D,\ \rho,\ \mu,\ V$

### Step 2: Express Each Variable in Terms of Base Dimensions

Use either MLT or FLT consistently.

| Variable | MLT | FLT |
|----------|-----|-----|
| $\Delta p_\ell$ | $ML^{-2}T^{-2}$ | $FL^{-3}$ |
| $D$ | $L$ | $L$ |
| $\rho$ | $ML^{-3}$ | $FL^{-4}T^2$ |
| $\mu$ | $ML^{-1}T^{-1}$ | $FL^{-2}T$ |
| $V$ | $LT^{-1}$ | $LT^{-1}$ |

### Step 3: Determine $r$ — The Number of Reference Dimensions

Count distinct base dimensions. For pipe flow: $F, L, T$ → $r = 3$

### Step 4: Select $r$ Repeating Variables

**Rules for selecting repeating variables:**
- Must contain **all $r$ base dimensions** among them
- Must be **dimensionally independent** (no one can be formed from the others)
- Should be **physically significant** and **easy to measure**

**Common choices for fluid mechanics:**

| Application | Repeating Variables | Reason |
|-------------|--------------------|---------|
| Pipe/internal flows | $D,\ V,\ \rho$ | Geometric scale, flow speed, fluid property |
| Free-surface flows | $L,\ V,\ \rho$ (or $g$) | Gravity important — include $g$ |
| Compressible flows | $L,\ V,\ \rho,\ c$ (or $E_v$) | Need speed of sound / bulk modulus |

For pipe flow: $D,\ V,\ \rho$ (contain $F,L,T$) — 3 repeating variables.

### Step 5: Form $\Pi$ Groups

For each remaining (non-repeating) variable, form a dimensionless product with the repeating variables:

$$\Pi_i = \text{(remaining var)} \times (\text{repeating vars})^{exponents}$$

Solve for exponents so dimensions cancel.

**Example — Pipe flow:**
- $\Pi_1 = \Delta p_\ell\, D^a V^b \rho^c$ → solve for $a,b,c$ to make $\Pi_1$ dimensionless
- $\Pi_2 = \mu\, D^a V^b \rho^c$ → same procedure

Result:
$$\Pi_1 = \frac{D \Delta p_\ell}{\rho V^2}, \qquad \Pi_2 = \frac{\rho V D}{\mu}$$

### Step 6: Express Result

$$\Pi_1 = \phi(\Pi_2, \Pi_3, \ldots)$$

or

$$\frac{D \Delta p_\ell}{\rho V^2} = \phi\!\left(\frac{\rho V D}{\mu}\right)$$

---

## 2. Checking Dimensionless Groups

Always verify that each $\Pi$ group is truly dimensionless:

$$\Pi = [\text{var}] \cdot [\text{rep}_1]^a [\text{rep}_2]^b [\text{rep}_3]^c$$

Set total exponent of each base dimension to zero, solve for $a,b,c$.

---

## 3. Non-Uniqueness of $\Pi$ Groups

The choice of repeating variables is **not unique** — different choices give different-looking $\Pi$ groups. However:

| Choice A | Choice B |
|----------|----------|
| $\Pi_1 = D\Delta p_\ell / \rho V^2$ | $\Pi_1' = \Delta p_\ell D / \mu V$ |
| $\Pi_2 = \rho V D / \mu$ | $\Pi_2' = \rho V D / \mu$ (same) |

Any $\Pi$ group can be replaced by:
- Its **product** with another $\Pi$ group
- Its **ratio** with another $\Pi$ group
- Any **power** of itself

All forms are mathematically equivalent.

## Key Takeaways

| Step | Action |
|------|--------|
| 1 | List $k$ variables |
| 2 | Write dimensions of each |
| 3 | Count $r$ = number of base dimensions |
| 4 | Select $r$ repeating variables |
| 5 | Form $k-r$ $\Pi$ groups |
| 6 | Write $\Pi_1 = \phi(\Pi_2, \ldots)$ |
